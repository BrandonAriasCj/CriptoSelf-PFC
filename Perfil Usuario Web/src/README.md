# CriptoSelf - Trading Cuantitativo Simplificado

Aplicación de trading algorítmico simplificada para crear, probar y ejecutar estrategias de trading cuantitativo.

## 🚀 Funcionalidades Principales

### ⚡ Estrategias
- **Construcción visual** de estrategias con triggers
- **5 tipos de triggers** disponibles: MA Cross, Volume Spike, RSI, Breakout, Bollinger
- **Configuración de parámetros** en tiempo real
- **Activación/desactivación** individual de triggers

### 📊 Backtesting
- **Gráfico interactivo** con visualización de triggers
- **Métricas de rendimiento** en tiempo real
- **Historial de eventos** detallado
- **Controles de reproducción** para análisis temporal

### 🎓 Formación
- **5 cursos especializados** en trading cuantitativo
- **Sistema de progreso** individual
- **Consejos prácticos** para principiantes
- **Estructura modular** fácil de seguir

### 📈 Actividad
- **Monitoreo en tiempo real** del bot
- **Estadísticas clave** de rendimiento
- **Historial de trades** recientes
- **Controles de bot** (pausar/reanudar)

## 🎨 Estructura Simplificada

```
/components
├── MyStrategy.tsx     # Construcción de estrategias
├── TradingChart.tsx   # Backtesting y análisis
├── Education.tsx      # Academia de trading
├── BotActivity.tsx    # Monitoreo del bot
└── Settings.tsx       # Configuración básica
```

## 📱 Características Técnicas

- **4 pestañas principales** para navegación simple
- **Tema oscuro** profesional
- **Completamente responsivo** para móviles/tablets
- **Componentes reutilizables** con ShadCN/UI
- **Datos simulados** para demostración
